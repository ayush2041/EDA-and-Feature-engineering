# EDA and Feature Engineering

A practical repository for learning and implementing Exploratory Data Analysis (EDA), Data Cleaning & Preprocessing, and Feature Engineering techniques with hands-on projects.

## 📚 Topics Covered

### Exploratory Data Analysis (EDA)
- Data visualization techniques
- Statistical analysis and pattern recognition
- Understanding data distributions and relationships

### Data Cleaning & Preprocessing
- Handling missing values
- Encoding categorical variables
- Feature scaling and normalization

### Feature Engineering & Selection
- Feature creation and transformation
- Feature selection methods
- Dimensionality reduction

## 🎯 Projects

- **Insurance Charges Prediction** - Predict medical insurance charges using regression techniques
- **Heart Disease Prediction** - Classify heart disease risk using classification models

## 🛠️ Tech Stack

- Python, Pandas, NumPy, Scikit-learn
- Matplotlib, Seaborn for visualization
- Jupyter Notebook

## 🚀 Getting Started

```bash
git clone https://github.com/ayush2041/EDA-and-Feature-engineering.git
pip install -r requirements.txt
jupyter notebook
```
